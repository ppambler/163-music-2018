# 仿网易云音乐

 包含管理端（admin 页面）
## 启动应用

  1. `npm i -g http-server`
 2. `http-server -c-1`
 3. `node server 8888`

ps：2和3是各自一个终端的，即你需要打开两个终端，然后定位到该项目目录，最后敲下这些指令即可

## 访问admin页面

open <http://127.0.0.1:8080/src/admin.html> 即可

其功能主要是关于数据库的操作，**✎：**

- 七牛云

  - 于本地上传歌曲到云端，为此得到歌曲外链

- leancloud

  歌曲数据——歌名、歌手、外链、封面、歌词

  - 添加歌曲数据
  - 查询歌曲数据
  - 编辑歌曲数据

## 访问index页面

open <http://127.0.0.1:8080/src/index.html> 即可

目前的功能只有简单的播放歌曲！
